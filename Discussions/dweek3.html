Week Three Discussion

Question:

Web design
In week 1, we read about HTML, CSS, Web design standards and practices, the principles of UX, and the Stages of web design. In week 2, we read about templates, and layouts and created our first webpage with the elements of a good web page. In week 3, we read about bootstraps, CSS and create our first website. What elements will you consider to design, construct and evaluate a web page? Remember to use at least 2 references in APA format to support your thoughts

My Discussion:

Having a responsive design helps ensure that the webpage is accessible on different devices and screen sizes. Bootstrap provides a grid system that automatically adjusts the layout based on the screen size. Media queries allow you to specify CSS rules that are applied only when certain conditions are met, such as the screen width. CSS also creates style for your webpage by interacting with HTML elements. CSS helps create an appealing and consistent visual design. CSS affects all aspects of your design from font, layout, color scheme to graphics and media. CSS also helps create an intuitive and user-friendly interface because it determines placement of items on your page, such as your navigation bar, buttons, forms and more. Having good placement and organization of information, or flow, creates a positive user experience.
